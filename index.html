Home Environment Monitor

About

This website functions as a fun way for me to learn about web development and the Internet of Things (IoT). I have set up several monitors throughout my apartment to track the temperature and humidity in each room, as well as monitor the moisture levels of my plants. I then track the readings obtained over time and display them below for temperature and humidity, and next to each plant for moisture levels on the Plants page.

You can find a detailed description of the project and its components in the Info section, and I post all the code that hosts this site on GitHub.
